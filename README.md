# 🛒 E-Commerce Sales Analysis using SQL

## Objective
To analyze e-commerce sales data and find key business 
insights using SQL queries.

## Tools Used
MySQL, MySQL Workbench

## Database Structure
- customers — 10 customers across major Indian cities
- products — 10 products across 4 categories
- orders — 20 orders with status tracking
- order_items — detailed order line items

## Queries Covered
1. Basic — COUNT, SUM, GROUP BY, ORDER BY
2. Intermediate — JOINS, HAVING, WHERE filters
3. Advanced — Window Functions, CTEs, Subqueries,
   CASE WHEN, Running Totals

## Key Business Findings
1. Total revenue from delivered orders — Rs 3,09,900
2. Electronics is the highest revenue category
3. Hyderabad customers generate most revenue
4. 2 orders were cancelled — 10% cancellation rate
5. Rahul Sharma and Suresh Babu are top customers

## SQL Concepts Used
SELECT, WHERE, GROUP BY, HAVING, ORDER BY,
INNER JOIN, LEFT JOIN, Subqueries, CTEs,
Window Functions — RANK, SUM OVER,
CASE WHEN, LIMIT, DATE Functions
